## installation

Build using `cargo build --release` from the directory.
Run using `visualiser <sorting algorithm>`. Sorting algorithm is case-insensitive.

Valid options for sorting algorithms are:
 - "insertion sort"
 - "selection sort"
 - "radix sort"
 - "merge sort
 - "stalin sort"
 - "bogosort"
 - "quantum bogosort"
 - "counting sort"
 - "odd-even sort"
 - "bubble sort"
 - "heap sort"
